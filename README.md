# Exercices R Débutant

Ce projet contient des exercices pour apprendre les bases du langage R.  
Il est destiné aux débutants et couvre la manipulation de vecteurs, le calcul de statistiques simples et le filtrage de données.

## Contenu du projet

- `exercices.R` : contient tous les exercices de base
- `environnement.RData` : sauvegarde de l'environnement R avec les vecteurs et variables créés

## Objectifs pédagogiques

1. Créer et manipuler un vecteur avec `c()`
2. Calculer la moyenne, le maximum et le minimum d’un vecteur avec `mean()`, `max()`, `min()`
3. Ajouter des éléments à un vecteur
4. Extraire des éléments spécifiques avec les crochets `[]`
5. Filtrer des valeurs selon des conditions logiques

## Résumé des exercices

1. **Créer un vecteur** de notes : 12, 15, 18, 10, 14
2. **Calculer la moyenne** des notes
3. **Ajouter une note** au vecteur
4. **Afficher la deuxième note**
5. **Afficher les notes supérieures à 14**

## Comment exécuter les exercices

1. Ouvrir `exercices.R` dans RStudio
2. Exécuter les lignes de code étape par étape
3. Vérifier les résultats affichés dans la console
